# FE 60关训练

一个给自己使用的日本基本情报技术者考试（FE）练习网页。直接打开 `index.html` 就能用，不需要服务器、不需要安装。

## 功能

- 科目 A、科目 B 各 30 关，每关 30 题
- 所有关卡自由选择，无需按顺序解锁
- 每题只分配到一个关卡，关卡之间不重复
- 每关显示答题完成百分比
- 中文和日语题面
- 手机友好的底部导航和触控布局
- PWA 支持：可添加到手机主屏幕，安装后可离线使用
- 错题本
- 学习统计
- 可选 OpenAI API Key 生成 AI 解释
- JSON 导入题库
- 所有学习数据保存在浏览器本地

## 手机上使用

现在已经是 PWA 版。手机第一次安装需要通过一个网址打开，安装后可以离线单独使用。

本地测试时，最简单的方法是让手机和这台 Mac 连接同一个 Wi-Fi，然后在这个文件夹里启动本地预览：

```text
python3 -m http.server 5173
```

再用手机打开：

```text
http://这台Mac的局域网IP:5173
```

如果只在 Mac 上模拟手机尺寸，直接缩窄浏览器窗口即可看到底部导航版布局。

正式长期使用建议把这个文件夹免费发布到 GitHub Pages、Cloudflare Pages 或 Netlify。手机访问发布后的 HTTPS 地址后：

- iPhone：Safari 分享按钮 → 添加到主屏幕
- Android：Chrome 菜单 → 安装应用

添加到主屏幕后，题库、错题和统计会保存在手机浏览器本地，并且页面会被缓存，断网也能打开。

## 导入题库格式

可以导入一个 JSON 数组：

```json
[
  {
    "id": "fe-2025-sample-001",
    "source": "IPA公开题",
    "year": "2025",
    "section": "科目A",
    "category": "网络",
    "title": "DNS",
    "text": "题目正文 / 問題文",
    "options": ["选项A", "选项B", "选项C", "选项D"],
    "answer": 1,
    "explanation": "简要解释"
  }
]
```

`answer` 从 0 开始，0 表示 A，1 表示 B。

关卡按科目和导入顺序自动分配，每 30 题组成一关。完整填满 60 关需要科目 A、B 各 900 道题；题目不足的关卡仍可进入，并会明确显示当前题量。只有提供真实 `year` 的题目才会标记为该年度真题，内置同等难度题不会冒充真题。

## 真题来源

IPA 官方过去问题页面：

https://www.ipa.go.jp/shiken/mondai-kaiotu/index.html

IPA 页面说明，公开的过去试题通常无需申请或使用费即可使用；同时也说明 2020-2022 年度的 SG/FE 问题非公开，2023 年以后有公开问题。导入前建议保留 `source` 和 `year`，方便以后按年份筛选。

## 关于 AI 解释

在“设置 / 設定”里保存 OpenAI API Key 后，练习页可以直接生成解释。Key 只保存在当前浏览器本地。若不想保存 Key，可以点击“复制提示词”，粘贴到 ChatGPT 使用。
